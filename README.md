# Blue Spot Analysis 2.0 — Documentación

Sitio de documentación del proyecto **Blue Spot Analysis 2.0 (BSA 2.0)** del Banco
Interamericano de Desarrollo. Cubre la metodología, la herramienta de escritorio
(toolbox de ArcGIS Pro) y el dashboard de resultados en línea.

Construido con [MkDocs](https://www.mkdocs.org/) + [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).
Desplegado en GitHub Pages: <https://andresabarca-atlas.github.io/bsa-docs/>

## Construir localmente

```bash
pip install -r requirements.txt
mkdocs serve
```

El sitio quedará disponible en `http://127.0.0.1:8000/`.

Para generar el sitio estático en la carpeta `site/`:

```bash
mkdocs build
```
